Find the Difference
(leetcode No. 389)

  • Tomoya Machida
  • Coding

This question is No. 389 Find the Difference . Here is the aproaches to slove this question.


There is two approch. 1. use dictionary and 2. use bit.

approch 1
1. prepare a dictionary and input elememnt of "t" into the dictionary with the number of element "t" has.
2. check if the letter in "t" is in "s" and if the dictionary has the elememnt, subtract one from the number which the key has. Once the number becomes zero, delete the key.
3. after 1, 2 step, you'll get dictionary which has only one key. and, turn the key into a string and return it. 

class Solution(object):
    def findTheDifference(self, s, t):
        dic1 = {}
        for i in t:
            if i not in dic1:
                dic1[i] = 1
            else:
                dic1[i] += 1

        for j in s:
            if j in dic1:
                dic1[j] -= 1
                if dic1[j] == 0:
                    dic1.pop(j)
        List = dic1.keys()
        return List[0]



approch 2
you can use the idea of bit apporch from Very Easy || 0 ms ||100%|| Fully Explained (Java, C++, Python, JS, C, Python3) of the question No. 136 Single Number .

the idea is if you use XOR toward everything you'll get the number which shows up only one time because XOR between same numbers is 0.


in the question, it uses string insted of number, so you need to turn the string into number with ord(). and turn the number into string with chr(). 

class Solution(object):
    def findTheDifference(self, s, t):
        result = 0
        for i in s:
            result ^= ord(i)
        for j in t:
            result ^= ord(j)
        return chr(result)
Time complexity: O(n)
Space complexity: O(1)
