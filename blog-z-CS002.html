Move Zeroes
(leetcode No. 283)

  • Tomoya Machida
  • Coding

This question is No. 283 Move Zeroes . Here is the aproach to slove this question.


1. roop the list. If the element of the list is not 0, then replace the element (i) with the element (num) from left to right.
("i" never overtake the element of the roop (num) or same as num at best.)



2. replace the element (i) with 0 until the end of the list.

Time complexity: O(n)
Space complexity: O(1)

class Solution(object):
    def moveZeroes(self, nums):
        i = 0
        for num in nums:
            if num != 0:
                nums[i] = num
                i += 1
        
        while i < len(nums):
            nums[i] = 0
            i += 1
        
        return nums
