Average of Levels in Binary Tree
(leetcode No. 637)

  • Tomoya Machida
  • Coding

This question is 637. Average of Levels in Binary Tree . Here is the aproaches to slove this question.


1. Like in the figure below, you'll update a queue every time it goes to new level of trees. when you go to a new level of trees, add up all numbers which the current level of tree has while you delete all the elements the queue has.

2. If the elements of queue has children, add the children to the queue.

3. Calculate the avarage of the queue.



Here is an example of the solution.
Let's say you're currently at second level.
Add up all numbers which the second level of tree has while you delete all the elements the secound queue has(root1, root2) by using pop(0).
Also, while deleting the elements, if the elements(root1, root2) has children, add the children to queue. (In this case only root1 has children.) 



class Solution(object):
    def averageOfLevels(self, root):
        deque, result = [root], []

        while deque:
            length, Sum = len(deque), 0 
            
            for _ in range(length):
                root = deque.pop(0)
                Sum += root.val
                if root.left: deque.append(root.left)
                if root.right: deque.append(root.right)
            result.append(Sum/float(length))

        return result

Time complexity: O(n)
Space complexity: O(1)
