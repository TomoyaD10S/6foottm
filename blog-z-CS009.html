Diameter of Binary Tree
(leetcode No. 543)

  • Tomoya Machida
  • Coding

This question is 543. Diameter of Binary Tree . Here is the aproaches to slove this question.


1. Add up the length of the longest path which root.left has and the length of the longest path which root.right has. if the number is more than 'ans', update 'ans'. (Note that 'ans' is nonlocal.)

2. In this recursion, return the maximum number between the length of the longest path which root.left has and the length of the longest path which root.right has.when you move to root.left, replace hi with root.val.

The figure blow are the steps to slove the question.

1-1 The length of the longest path which root.left has is 2. the length of the longest path which root.right has is 1. So, the total is 3. Since the number is more than 'ans'(1 at this moment), update the 'ans'.



1-2 Since the length of the longest path which root.left has is longer than the length of the longest path which root.right has, return the length of the longest path which root.left has.



1-1 The length of the longest path which root.left has is 3. the length of the longest path which root.right has is 2. So, the total is 5. Since the number is more than 'ans'(3 at this moment), update the 'ans'.



1-2 Since the length of the longest path which root.left has is longer than the length of the longest path which root.right has, return the length of the longest path which root.left has.



Python 

class Solution(object):
    def diameterOfBinaryTree(self, root):
        dic={0:0}
        def recursion(root):
            if not root:
                return 0
            left = recursion(root.left)
            right = recursion(root.right)
            dic[0]=max(dic[0],left+right)
            return 1+max(left,right)
        recursion(root)
        return dic[0]

Python3 
class Solution:
    def diameterOfBinaryTree(self, root: Optional[TreeNode]) -> int:
        ans=0
        def dia(root):
            nonlocal ans
            if not root:
                return 0
            left=dia(root.left)
            right=dia(root.right)
            ans=max(ans,left+right)
            return 1+max(right,left)
        dia(root)
        return ans

Time complexity: O(n)
Space complexity: O(1)
